# Boss 个人知识库

这个知识库采用轻量 PARA + 卡片沉淀法：

- `00_Inbox`：先收集，不纠结分类
- `01_Projects`：有截止时间、要交付的事项
- `02_Areas`：长期负责的领域（工作/健康/学习/家庭等）
- `03_Resources`：可复用资料（方法、模板、参考）
- `04_Archive`：已完成/低活跃内容
- `templates`：标准化模板

## 使用节奏（建议）

### 每天（5-10分钟）
1. 把碎片想法放进 `00_Inbox`
2. 把高价值条目转成卡片（用模板）

### 每周（20-30分钟）
1. 清空 Inbox（归档到 Projects/Areas/Resources）
2. 复盘本周 3 个最有价值结论
3. 给下周生成行动清单

## 命名建议

- 知识卡：`YYYY-MM-DD-主题关键词.md`
- 项目页：`项目名.md`
- 周复盘：`weekly-YYYY-WW.md`

## Ug 协作方式

你可以直接对 Ug 说：
- “把这段整理进知识库（项目X）”
- “把今天聊天提炼成3条卡片”
- “生成本周复盘与下周行动”
